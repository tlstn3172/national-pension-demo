# A1. 계산 로직 상세

## 기금 잔액 계산 공식

```javascript
기금 잔액(t+1) = 기금 잔액(t) × (1 + 수익률) 
                + 보험료 수입(t) 
                - 연금 지급액(t)

보험료 수입(t) = 가입자 수(t) × 평균 소득(t) × 보험료율

연금 지급액(t) = 수급자 수(t) × 평균 소득(t-n) × 소득대체율
```

**변수 설명**
- t: 연도
- n: 가입 기간 (일반적으로 20-30년)
